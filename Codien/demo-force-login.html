Demo Force Login Popup

Các tính năng bắt buộc đăng nhập

Force Login

Popup không thể tắt bằng nút X, click outside, hoặc ESC key

Normal Login

Popup có thể tắt bình thường

Check Auth Status

Kiểm tra trạng thái đăng nhập hiện tại

Logout

Đăng xuất và hiển thị force login nếu cần

Cách sử dụng
1. Force Login (Không thể tắt):
const loginPopup = new LoginPopup({
    forceLogin: true,
    checkAuthOnInit: true
});
2. Normal Login (Có thể tắt):
const loginPopup = new LoginPopup({
    forceLogin: false
});
3. Global Functions:
// Force login
showForceLoginPopup();

// Check auth
checkAuth();

// Normal login
showLoginPopup();
Tính năng Force Login
  • Không có nút tắt (X) - User không thể đóng popup
  • Không thể click outside - Backdrop static
  • Không thể dùng ESC - Keyboard disabled
  • Tự động kiểm tra auth - Hiển thị popup nếu chưa đăng nhập
  • Kiểm tra token hết hạn - Tự động logout nếu token expired
